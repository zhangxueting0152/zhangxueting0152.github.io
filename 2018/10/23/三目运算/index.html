三目运算

语法：

基本语法：expression ? sentence1 : sentence2
当expression的值为真时执行sentence1，否则执行sentence2。

相当于if else:

当需要进行多次逻辑判断时，三目运算符会看起来更加的简洁：

可以看到，只要任意一个expressionN为真，就会立即执行sentenceN,并且判断结束，不会执行后面的代码。对应的if else如下：

相比之下，三目运算符更加的简洁。

应用

1.判断

2.赋值

三目运算是有返回值的，相当于return(3)

所以当写成var c = (a >= 1 ? return true : return false);的时候，就会有问题，因为这个相当于
if(expression){
return (return true);//会报错
}

因为b++是后加，所以return b++,是先return b,再给b加1的。

&&

假前真后：
只要&&前面是false，无论&&后面是true还是false，结果都将返回&&前面的值；
只要&&前面是true，无论&&后面是true还是false，结果都将返回&&后面的值；
