CSS隐藏元素

1) opacity

视觉上它是消失了,但是他依然占据着那个位置。它也将响应用户交互。

2) display：none

该元素会从视觉中消失,并且连盒模型也不生成.也不会在页面占据任何位置。不响应交互。

3) visibility:hidden

元素将会隐藏，也会占据着自己的位置，并对网页的布局起作用。不会响应任何用户交互。

4) position：absolute

视觉上它是消失了,但是他依然占据着那个位置。它也将响应用户交互。
